# house-price-prediction-ml
IA qui prédit le prix d’une maison
Machine learning project that predicts house prices using features such as:

- number of bedrooms
- number of bathrooms
- living area size

## Technologies

- Python
- Pandas
- Scikit-learn
- Matplotlib

## Dataset

Kaggle House Sales dataset.

## How to run

Install dependencies:

pip install -r requirements.txt

Train model:

python src/train.py

Make prediction:

python src/predict.py
